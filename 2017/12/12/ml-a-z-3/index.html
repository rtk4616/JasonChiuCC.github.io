Simple Linear Regression

線性回歸(Simple Linear Regression)

y = b0 + b1*x1這代表一直線

y = Dependent variable (DV) = 應變量

x = Independent variable (IV) = 自變量

y 是隨著 x 變化的

b1 = Coefficent (斜率，與此直線的角度有關)

b0 = Constant (常數，y 截距)

原始數據如下，分別為工作時間與工資

用上述公式畫出圖形如下

Salary = b0 + b1* Experience

b0 = 紅色圈起來的部分，也就是當年資是 0 時可以領到的薪水

而 b1 是年資與薪水的一種關係

表示當年資增加時，薪水會增加多少

如上圖綠色標示，當年資增加一年，薪水會增加 10k

這邊有個專有名詞”擬合”

擬合表示透過機器學習最後得到此直線的”過程”

簡單來說，就是要找出 b0 和 b1 這兩個參數

並且這兩個參數會讓此直線有最好的預測結果

ŷi 稱為 yi hat or hat yi (y 戴帽子)

通常變數加上hat 就等於此變數是利用模型預測出來的結果

也就是說 yi 是真實的數據資料

而 ŷi 只是機器預測出來的值而已

有此可知，如果機器預測出來的值和真實數據越接近時

表示此機器預測的很準，所以可以用下面公式來表示
