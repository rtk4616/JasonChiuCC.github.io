Hexo 貼圖教學

Hexo 版本

貼圖方法

這是 Hexo 3.0 新增的貼圖方法，稱為”資產資料夾(asset_folder)”

傳統貼圖方法是你需要將圖片上傳到某個免費空間，之後再用 Markdown 語法輸入圖片 URL 做連結

而 asset_folder 方法是直接將圖片和文章放在一起，這樣管理起來也比較方便
