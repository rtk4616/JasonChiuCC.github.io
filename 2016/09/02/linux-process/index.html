執行 Process

將 Process 在背景執行加上 &

查看目前背景 Process 輸入 jobs

將背景程式喚回前景，使用 fg 後面加序號或 PID

如果要再放回背景按下 Ctrl+Z

不過按下 Ctrl+Z 之後在背景目前是暫停的

要讓他在背景繼續執行要輸入 bg

使用 process name 刪除 process
